# Tooo Excel处理工具 v1.0.0 发布说明

## 🎉 版本亮点

### 核心功能
- ✅ **Excel文件拆分**: 支持按行数拆分大型Excel文件
- ✅ **Excel文件合并**: 支持合并多个Excel文件为一个
- ✅ **格式保留**: 提供基础版和格式保留版两种处理模式
- ✅ **多格式支持**: 兼容.xlsx和.xls格式文件
- ✅ **表头处理**: 灵活的表头复制和去重选项

### 技术特性
- 🚀 **现代界面**: 基于React + Electron的桌面应用
- 🎨 **美观设计**: 使用Tailwind CSS构建的现代化UI
- 🐍 **Python后端**: 使用pandas和openpyxl处理Excel文件
- 📱 **响应式设计**: 适配不同屏幕尺寸
- 🔧 **易于使用**: 直观的操作界面，无需技术背景

## 📦 下载和安装

### 便携版（推荐）
- **文件名**: `Tooo-Portable-v1.0.0.zip`
- **大小**: 约110MB
- **系统要求**: Windows 10/11 x64
- **安装方式**: 解压即用，无需安装

### 使用步骤
1. 下载 `Tooo-Portable-v1.0.0.zip`
2. 解压到任意目录
3. 双击 `Tooo.exe` 启动应用
4. 开始处理Excel文件

## 🔧 功能说明

### Excel拆分功能
- 支持按指定行数拆分Excel文件
- 可选择是否在每个拆分文件中复制表头
- 提供基础版（快速）和格式保留版（完整格式）
- 自动生成带序号的输出文件名

### Excel合并功能
- 支持合并指定目录下的所有Excel文件
- 可选择是否去除重复表头
- 保持原始数据完整性
- 支持混合.xlsx和.xls格式文件
